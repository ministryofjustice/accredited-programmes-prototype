Shortcut Keys

You can use shortcut keys to navigate more quickly and easily. Use the listed keys in the stated places.

Home Screen

  • + Q - National Search
  • + N - National Custody Search
  • + R - Recently Viewed
  • + U - UPW Project Diary
  • + J - UPW Projects
  • + F - Officer Diary
  • + C - Court Diary

Home Screen

  • + A - AP Diary
  • + Z - User Preferences
  • + X - Sign Out
  • + S - New Search
  • + V - Create and Vary

Header and Menu

  • + H - Home Page
  • + F - Officer Diary
  • + P - Print
  • + K - Alerts Diary
  • + V - Register Reviews
  • + X - Exit
  • + Y - My Documents
  • + Z - User Preferences
  • + 0 - Toggle Header View

On Screens

  • + A - Add
  • + C - Close / Return
  • + C - Cancel
  • + D - Document
  • + D - Delete
  • + D - Deselect All
  • + E - Edit
  • + E - Clear
  • + L - Spell Check

On Screens

  • + O - Confirm
  • + Q - Search
  • + R - Transfer
  • + R - Print
  • + S - Save
  • + S - Select All
  • + U - Update
  • + 1 - Focus on first input field
